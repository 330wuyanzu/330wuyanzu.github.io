    渗漏检查-液压类

    330吴彦祖 2022/1/3 A320FM渗漏液压飞控起落架货舱门
    参考 AMM 29-00-00-790-001-A

    术语(3种渗漏类型)

    • leak : 大量的液压油从部件里渗漏出来，并且可以滴下来或者即将滴下来(`20滴 ≈ 1cc`，`75600滴 ≈ 1加仑`)
    • leakage : 大量的液体出现在部件表面，但不会滴下来
    • stain : 指的是部件表面上有颜色不一样的区域，通常是由于渗漏的液压油被高温烤干后导致的

    NOTE

    正常刹车系统不允许液压渗漏，因为会影响适航安全

    先看部件级的渗漏是否超标，再看系统级(总体)的渗漏是否超标

    # A、检查部件级的渗漏是否超标（不适用于备用刹车低压系统）

    NOTE

    • 大多数的部件渗漏需要在增压的情况下统计分析
    • 增压之前必须擦干净部件表面的油液、油迹、油痕
    • 大多数的部件在增压后还应进行多次的操作作动才能正确统计分析渗漏情况

    NOTE 6 总体原则：

    如果一个或多个部件外部渗漏超过正常标准，飞机依然可以放行，但必须满足以下条件：

    • 渗漏没有超过放行限制
    • 只能有一个液压系统出现部件渗漏超过正常限制
    • 对于给定的液压系统，渗漏必须小于放行限制
    • 液压油箱里的油量(液面高度)指示正确

    放行后，有必要在每次飞行前再次执行相应检查

    # A.1、液压泵类

    适用部件

    • EDP: Engine Driven Pump
    • EMP: Electric Motor Pump
    • RAT: Ram Air Turbine
    • PTU: Power Transfer Unit
    位置 正常情况下渗漏标准 放行(保留)标准
    静封严(管接处、连接处) 不允许渗漏 连续10min 内 2滴
    静套管(壳体) 连续10min 内 1滴 1滴/min
    轴封严(增压时) 2滴/min 8滴/min
    EDP 轴封严(作动时) 5滴/min 60滴/min
    EMP 轴封严(作动时) 5滴/min 30滴/min
    RAT 轴封严(作动时) 1滴/min 1滴/min
    PTU 轴封严(作动时) 5滴/min 30滴/min

    # A.2、伺服控制类（操纵面）

    适用部件

    • 升降舵
    • 副翼
    • 方向舵
    • 扰流板
    • 偏航阻尼器

    NOTE 1

    对于升降舵、副翼：同一操纵面不允许有两处渗漏超过正常标准

    NOTE 4

    对于偏航阻尼器伺服作动器：

    • (-∞, 4)滴/min: no more steps are necessary
    • [4, 14]滴/min: 在750FH内要更换偏航阻尼器伺服作动器
    • [15, 30]滴/min: 在100FH内要更换偏航阻尼器伺服作动器
    • (30, +∞)滴/min: 必须立即更换
    • 使用件号SC4700-3目视检查偏航阻尼器作动器AMM TASK 27-26-51-200-001
    位置 正常情况下渗漏标准 放行(保留)标准
    静封严(增压时) 不允许渗漏 连续10min 内 2滴
    动封严(增压时)-升降舵 连续10min 内 1滴 连续10min 内 6滴
    动封严(增压时)-副翼 连续10min 内 1滴 连续10min 内 6滴
    动封严(增压时)-方向舵 连续10min 内 1滴 30滴/min
    动封严(增压时)-扰流板 连续10min 内 1滴 30滴/min
    动封严(作动时)-升降舵 连续10个循环 1滴 连续10个循环 3滴
    动封严(作动时)-副翼 连续10个循环 1滴 连续10个循环 3滴
    动封严(作动时)-方向舵 连续10个循环 1滴 15滴/循环
    动封严(作动时)-扰流板 连续10个循环 1滴 15滴/循环

    # A.3、作动器类

    例如

    • 起落架作动器
    • 起落架舱门作动器
    • 货舱门作动器
    • 自锁式作动器
    • 等等。。。。。

    NOTE 7

    2 drops in 10 min 是针对货舱门增压和操作时的标准

    位置 正常情况下渗漏标准 放行(保留)标准
    静封严 不允许渗漏 连续10min 内 2滴
    活塞杆封严(增压时) 1滴/min 30滴/min
    活塞杆封严(作动时) 1滴/循环 2滴/循环

    # A.4、管路连接类

    NOTE 2 强制性要求

    前提是连接处必须紧，且打了正确的力矩

    位置 正常情况下渗漏标准 放行(保留)标准
    可拆装的pipe to pipe连接 不允许渗漏 连续10min 内 2滴
    永久性的pipe to pipe连接 不允许渗漏 不允许渗漏
    swivel fitting 不允许渗漏 30滴/min
    歧管fitting 不允许渗漏 连续10min 内 2滴
    bobbin 不允许渗漏 连续10min 内 2滴
    可拆装的pipe to fitting 不允许渗漏 连续10min 内 2滴
    sliding unions compensated slide slight leakage 连续10min 内 1滴

    # A.5、其他

    NOTE 3 强制性要求

    • 备用刹车系统里不得有部件或连接处(安全活门、歧管、等等。。)渗漏超过 2滴/10min
    • 备用刹车系统里包含的所有部件和连接处的总的渗漏率不得超过 10滴/min

    NOTE 5

    备用刹车-刹车组件(设置停留刹车时)：如果温度小于-40℃，则可以允许 10滴/min

    位置 正常情况下渗漏标准 放行(保留)标准
    静封严 不允许渗漏 连续10min 内 2滴
    动封严(增压时) 不允许渗漏 1滴/min
    液压马达轴封严(襟缝翼、THS)增压和作动时 5滴/min 30滴/min
    CSM/G 1滴/min 6滴/min
    备用刹车-刹车组件(休息时) 不允许渗漏 不允许渗漏
    备用刹车-刹车组件(设置停留刹车时) 1滴/min 2滴/min
    地面渗漏测量歧管(休息时) 2滴/天 4滴/天
    地面渗漏测量歧管(增压时) 连续5个循环 1滴 连续5个循环 2滴
    地面渗漏测量歧管(增压和作动之后) 连续15min 内 1滴 连续15min 内 2滴
    手摇泵(货舱门、RSVR Filling) 5滴/min 30滴/min
    优先活门 不允许渗漏 连续10min 内 2滴
    人工选择活门 2滴/100个操作 2滴/10min
    液压油箱加油管路 不允许渗漏 不允许渗漏

    # A.6

    NOTE

    • If no external hydraulic leak is found, it is possible that the leak is in the fuel tank(s)
    • Do a fuel sampling for hydraulic fluid contaminationAMM TASK 12-32-28-281-802
    • If hydraulic fluid is found in the fuel tank(s), contact AIRBUS

    # B、再检查系统级的渗漏是否超标

    # B.1、正常渗漏检查标准

    以下条件必须同时满足：

    1. 绿、黄、蓝三个液压系统总的渗漏率不得超过 50滴/min
    2. 如果在接下来的目的地机场的温度小于-20℃，则必须将相应的液压油箱加油至最大油量位：
      • 绿系统：16.0 = 14.0+2.0
      • 黄系统：14.0 = 12.0+2.0
      • 蓝系统：7.0 = 6.0+1.0
    3. 渗漏的液压油不得滴到带电用电设备上

    # B.2、放行标准

    以下条件必须同时满足：

    1. 单个系统渗漏率在[50, 100]滴/min
    2. 相应系统的油箱加满：
      • 绿系统：14.5
      • 黄系统：12.5
      • 蓝系统：6.5
    3. 部件级渗漏率必须小于放行标准
    4. 只能有一个系统存在部件级渗漏超过 50滴/min
    5. 渗漏的液压油不得滴到带电用电设备上
    6. 每次飞行前必须检查渗漏情况，且必须满足以上5个条件
    7. 三个系统总的渗漏率不得超过 100滴/min

    如果是液压储压器充气管路渗漏，尝试增压以停止渗漏AMM TASK 12-14-29-614-001，如果不行需更换相应部件
