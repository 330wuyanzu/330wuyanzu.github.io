    ATA-31：Indicating & Recording

    330吴彦祖 2023/3/14 A320FM指示/记录系统

    A320FM

    ATA-31章主要讲解飞机上(主要是驾驶舱)的仪表和数据记录系统，这些系统不仅仅用于飞行员操作控制以及监控飞机状态， 它们更像是飞机的神经和大脑一样，给飞机身体的各个部分传递控制信号，也收集各个部分的反馈，同时存储和分析这些数据。

    # 简介 FIXME

    ATA-31章的核心内容可以划分为以下几大部分：

    • 独立仪表系统，即电子时钟
    • 仪表系统之电子中央飞机监控系统
    • 仪表系统之电子飞行仪表系统
    • 数据记录系统之中央警告系统，
    • 数据记录系统之中央故障显示系统
    • 数据记录系统之飞行数据记录系统(AIDS、CVR、FDR等等)

    电子时钟是飞机的时钟源，

    # 一、ATA 31-21 电子时钟

    • FIN: 2FS
    • 数量： 1个
    • 安装位置： 驾驶舱舱前方主仪表板中央（EWD右边，起落架收放手柄上方）
    • 作用： 飞机的时间基准
    • 用户(接口)： 两部FMGC、FDIU、CFDIU
    • 备份： 失效时，CFDIU内部有一个内锁电路时钟作备用时间基准，需要通过MCDU设置时间
    • 主供电： 28VDC
      • 正常供电，重要电路汇流条401PP，跳开关6FS - 头顶面板49VU
      • 备用供电，热电瓶汇流条701PP，跳开关4FS - 后面板121VU
    • 灯光供电： 5VAC， 驾驶舱灯光控制16LF(操作开关33LP - 头顶面板25VU)

    电子时钟的内部有时基(电路)，但通常会将开关打到GPS位置，这样可以使用GPS授时(GPS是原子钟，精度很高)，同时来自GPS的时间会同步到时钟的内部时基， 因此当GPS信号丢失或人工切换到INT位置时，时钟会通过其内部的时基继续运行，而不会丢失时间，当然也可以打到SET位，人工设置内部时基的时间

    # 1、功能

    • 基本的时间、日期功能(UTC)：时、分、秒、年、月、日
    • 计时
    • 秒表功能(在两侧的ND的上显示，FCU两侧各有一个按钮用于开始、暂停、重置)

    # 2、数据

    • 数据协议：

      • 时分秒： BCD编码(标号125)、BNR编码(标号150)
      • 年月日： BCD编码(标号260)
      • 设备标识(0x31): 16进制编码(标号377)

    • 通信协议： ARINC 429，ARINC 429以字为单位传输数据，ARINC 429的一字为四字节，即32位，每个字中的标号表示该字所传输的是什么数据， 时间数据按以下顺序传输，每32.25ms传输一个参数，因此传输一次时间共需125ms

      1. LABEL 125
      2. LABEL 150
      3. LABEL 260
      4. LABEL 377

    # 3、ARINC 429粗略介绍

    ARINC 429字的编码结构：

    • 1-8位： LABEL，八进制的标号，用于表示所传输的数据的意义(数据类型)
    • 9-10位： SDI，用于表示数据的来源或数据的终端识别码
    • 11-28或29位： 与数据类型有关，用于表示所传输的数据
    • 29或30-31位： 与数据类型有关，用于表示所传输数据的特性或状态
    • 第32位： 奇偶校验位，用于减少数据传输误差率

    ARINC 429的字中，后24位为符合阅读习惯的大端序，前8位LABEL类似小端序，是从右往左读的

    # 4、时钟数据示例解析

    1. 第一个字，LABEL 125，8421BCD编码的时分秒，UTC 23点59分54秒：
    • LABEL，o125(注意端序): b01010101 == 0x55 == 85 == o125
    • 数据，23:59:54：
      • 27-29位，小时的十位 8421码(0)010， 表示十进制的2
      • 23-36位，小时的个位 8421码0011，表示十进制的3
      • 19-22位，分钟的十位 8421码0101，表示十进制的5
      • 15-18位，分钟的个位 8421码1001，表示十进制的9
      • 11-14位，分钟整除10？ 8421码1001，表示十进制的9
    1. 第二个字，LABEL 150，BNR编码的时分秒，UTC 23点59分59秒：
    • LABEL，o150(注意端序): b01101000 == 0x68 == 104 == o150
    • 数据，23:59:59：
      • 24-28位，小时， 二进制编码b(000)10111 == 0x17 == 23
      • 18-23位，分钟，二进制编码b(00)111011 == 0x3B == 59
      • 12-17位，秒，二进制编码b(00)111011 == 0x3B == 59

    1. 第三个字，8421编码的年月日，UTC 86年12月25号，参考第一个字

    2. 第四个字，设备识别码，0x31

    • 15-18位十位，3
    • 11-14位个位，1

    # 5、故障现象

    • 时钟故障： 没有显示
    • 失去正常供电：
      • 没有显示
      • 但是有备用供电(电瓶)时，内部仍然会走时
      • 不会对外输出数据
      • 时钟上的按钮不可用
    • 失去备用供电： 直接关机了，但只要恢复供电即可通过GPS恢复走时
    • 同时失去两个供电： 完全挂了
