    雷达罩(石英、凯夫拉)损伤检查

    330吴彦祖 2021/12/19 A320FM雷达罩
    参考 AMM 53-15-11-200-001-A
适用带外部导电条的型号(凯夫拉或石英材料)

    # A、雷达罩区域划分和损伤区域计数

    B区除掉漆外，不允许其他类型的损伤

    # A.1、第一步：X-任意两个相邻损伤的距离

    • 若 X＞300mm，这两个损伤应当视为不同的损伤区域
    • 若 X＜300mm，且 X ≥ maxof(dia.I, dia.J)，这两个损伤视为不同的损伤区域
    • 若 X＜300mm，且 X＜ maxof(dia.I, dia.J)，这两个损伤视为同一个损伤区域

    # A.2、第二步：Y-新损伤与已修理损伤之间的距离

    • 任意一个新损伤和任意一个已修理损伤之间的距离 Y不能小于150mm
    • When it is not possible to measure minimum distance Y, you must count the adjacent areas as only one area.有歧义
    • 若允许的修理区域数量已达上限，则新的修理区域必须将已修理区域包含在内(即可以视为一个修理区域的情况)，如果做不到，则必须更换雷达罩

    # A.3、第三步：C-A区损伤到B区的距离

    若 C ≥ 0mm，则可以根据损伤类型进行临时修理， 否则立即更换雷达罩

    # B、外表面损伤

    外表可能出现的损伤类型如下：

    • scratch：划痕、划伤、刮痕、刮伤
    • gouge：凹槽、凿痕、凿出的槽、孔、洞
    • nick：裂口、刻痕
    • delamination(纤维不能断裂)：分层
    • debonding(纤维不能断裂)：剥离、脱粘、脱胶
    • core crushing/dent(纤维不能断裂)：凹坑
    • perforation：穿孔
    • paint peeling：掉漆
    • 单层结构损伤(B区)

    在用胶带临时修理前，必须保证已有的胶带面积不得超过490cm²，临时修理后，所有的胶带总面积也不得超过490cm²

    # B.1、刻划类损伤(划伤、凿伤、裂口、刻痕)

    • 首先，损伤直径不能大于 250mm，否则更换雷达罩
    • 其次，按前文对所有刻划损伤划区：
      • 若损伤区域里的损伤的最大直径 ≤50mm，这样的损伤区域最多允许有5个
      • 若损伤区域里的损伤的最大直径 (50, 250]mm，这样的损伤区域最多允许1个
    • 最后：
      • 若只有一层结构有损伤，则立即用胶带临时修理，保留600FH永久修理
      • 否则更换雷达罩

    # B.2、掉漆

    • 所有的掉漆总面积不得超过490cm²，否则更换雷达罩
    • 立即用胶带临时修理，保留600FH喷漆
