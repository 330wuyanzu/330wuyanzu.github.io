    雷达罩损伤检查

    330吴彦祖 2021/12/19 A320FM雷达罩
    参考 AMM 53-15-11-200-001-A
FIN 1560WM

    • 确认损伤区域
    • A、B区损伤
    • B、A区-外表面损伤
    • B.1、初步判断，确认损伤数量
      • B.1.a: 1st 测量任意两个相邻损伤区域之间的距离
      • B.1.b: 2nd 测量损伤和已临时修理区域的距离
      • B.1.c: 3rd 测量损伤到B区的距离
    • B.2、检查确认损伤类型，有数量限制
      • B.2.a: 划伤、凿伤、裂口、刻痕
      • B.2.b: 掉漆
      • B.2.c: 分层、脱胶、凹坑
      • B.2.d: 烧蚀(雷击)-导电条、开关手柄
      • B.2.e: 脱胶-导电条

    NOTE 适用性

    • 凯夫拉纤维材料的雷达罩：
      • PN D5311047700001
      • PN D5311047700002
      • PN D5311047700003
      • PN D5311047700004
      • PN D5311047700005
    • 石英纤维材料的雷达罩：
      • PN D5311047700006
      • PN D5311047700007
      • PN D53132010000

    # 确认损伤区域

    # A、B区损伤

    B区仅允许掉漆(参照B.2.b: 掉漆)，否则需要更换雷达罩

    # B、A区-外表面损伤

    NOTE

    已有的用胶带临时修理的区域和即将用胶带临时修理的区域，它们的胶带的总面积不得超过490cm²，否则进行永久性修理或者更换雷达罩

    # B.1、初步判断，确认损伤数量

    # B.1.a: 1st 测量任意两个相邻损伤区域之间的距离

    • 如果 X ＞300mm，这两个区域视为不同的损伤区域
    • 如果 X ＜300mm，且 ≥ maxof(dia.I, dia.J)，这两个区域视为不同的损伤区域
    • 如果 X ＜300mm，且 ＜ maxof(dia.I, dia.J)，这两个区域视为同一个区域

    # B.1.b: 2nd 测量损伤和已临时修理区域的距离

    • 任意一个已有临时修理区域和任意一个新损伤区域的距离 Y 不得＜150mm after tap test
    • 如果无法测量 Y，则必须将相邻的区域当作同一个区域
    • 如果已经达到允许临时修理区域的最大数量，则新损伤的修理区域必须包含已修理区域(即可以视作同一修理区域的情况，数量就不会超限)，如果做不到，则必须更换雷达罩

    # B.1.c: 3rd 测量损伤到B区的距离

    • 如果 C ≥ 0mm，则可以根据损伤类型进行临时修理， 否则立即更换雷达罩

    # B.2、检查确认损伤类型，有数量限制

    损伤类型：

    • scratch：划痕、划伤、刮痕、刮伤
    • gouge：凹槽、凿痕、凿出的槽、孔、洞
    • nick：裂口、刻痕
    • delamination(纤维不能断裂)：分层
    • debonding(纤维不能断裂)：剥离、脱粘、脱胶
    • core crushing/dent(纤维不能断裂)：凹坑
    • perforation：穿孔
    • paint peeling：掉漆
    • 单层结构损伤(B区)

    # B.2.a: 划伤、凿伤、裂口、刻痕

    NOTE 数量限制

    只允许出现：

    • 1个最大直径 ≤250mm的损伤(有可能会有视作同一个损伤的多个损伤)
    • 或者 5个最大直径 ≤50mm的损伤(有可能会有视作同一个损伤的多个损伤)
    • 如果直径 ＞250mm，立即更换雷达罩
    • 如果直径 ≤ 250mm：
      • 如果只有一层材料损伤，立即临时修理AMM TASK 53-15-11-300-001，可以保留600FH
      • 如果超过一层材料损伤，立即更换雷达罩

    # B.2.b: 掉漆

    NOTE

    注意不要忘记损伤总面积不得超过 490cm²的前提

    • 如果最少有一层掉漆，应当立即进行临时修理，可保留600FH

    # B.2.c: 分层、脱胶、凹坑

    NOTE 数量限制

    只允许出现：

    • 1个最大直径 ≤250mm的损伤(有可能会有视作同一个损伤的多个损伤)
    • 或者 5个最大直径 ≤50mm的损伤(有可能会有视作同一个损伤的多个损伤)
    • tap testAMM TASK 53-15-11-700-002，确认损伤直径
    • 如果损伤直径 ≤50mm，立即临时修理，可保留600FH后永久修理
    • 如果损伤直径 (50, 250)mm，立即临时修理，可保留600FH后更换雷达罩
    • ≥250mm，立即更换雷达罩

    # B.2.d: 烧蚀(雷击)-导电条、开关手柄

    • 导电条上的螺钉烧蚀：无需临时处理，保留600FH
    • 导电条上的烧蚀：无需临时处理，保留600FH
    • 开关手柄上有烧蚀、烧熔的区域，可保留600FH，但必须同时满足以下条件：
      1. 可以正常关锁
      2. 可以正常上锁
      3. 可以正常开锁
      4. 锁的张力没问题且可以调节
      5. 间隙、富余度、steps正常
